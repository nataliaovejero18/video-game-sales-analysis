# Video Game Sales Data Analysis

## Project Overview
This project analyzes global video game sales data to identify trends and patterns across different regions, platforms, and genres.

## Objectives
- Understand sales distribution by region
- Identify top-performing platforms and genres
- Analyze sales trends over time

## Methodology
- Data cleaning and preparation
- Exploratory data analysis
- Data visualization using BI tools

## Tools
- Power BI
- Excel
- Data visualization

## Key Insights
- Sales vary significantly across regions
- Certain platforms and genres dominate the market

## What I learned
This project helped me improve my data visualization skills and understand how to present insights clearly for business decision-making.
